## 📖 Proje Hakkında

Fem Davet; düğün, nişan, söz, kına, doğum günü ve kurumsal etkinlikler gibi özel gün organizasyonları düzenleyen bir işletmenin dijital yüzüdür. Bu proje, markanın vizyonunu yansıtacak zarif bir tasarımla, kullanıcı dostu ve yüksek performanslı bir web deneyimi sunmak amacıyla uçtan uca tasarlanmış ve kodlanmıştır. 

Ziyaretçilerin aradıkları bilgiye en hızlı şekilde ulaşabilmesi için modern "Tek Sayfa" mimarisiyle kurgulanmış olup, herhangi bir kütüphane veya framework bağımlılığı olmadan saf (vanilla) web teknolojileriyle geliştirilmiştir.

## 🚀 Öne Çıkan Özellikler

* **Tam Duyarlı (Responsive) Tasarım:** Mobil, tablet ve masaüstü cihazlarla %100 uyumlu, CSS Flexbox ve Grid sistemleri ile inşa edilmiş esnek arayüz.
* **Akıcı Gezinme (Smooth Scrolling):** Tek sayfa yapısı içerisinde, JavaScript destekli menü yönlendirmeleriyle sayfa içi bölümlere yumuşak geçiş imkanı.
* **Kapsamlı SEO Optimizasyonu:** Google aramalarında (özellikle yerel aramalarda) öne çıkmak için anlamsal (semantic) HTML5 yapısı, Schema.org (LocalBusiness) işaretlemeleri ve Open Graph etiketleri entegrasyonu.
* **Hızlı İletişim (CTA) Dönüşümü:** Ziyaretçileri müşteriye dönüştürmeyi hedefleyen, doğrudan WhatsApp yönlendirmesi ve tek tıkla arama butonları.
* **Dinamik Medya Gösterimi:** Optimize edilmiş fotoğraf ve HTML5 video bileşenlerini barındıran asimetrik galeri düzeni.

## 💻 Kullanılan Teknolojiler ve Araçlar

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (DOM Manipülasyonu)
* **Tasarım & UI:** CSS Custom Properties (Değişkenler), Google Fonts (Inter & Playfair Display), FontAwesome Icons
* **Dağıtım (Deployment) & Ağ:** Git, GitHub Pages, Özel Alan Adı (Custom Domain) Entegrasyonu, DNS Yönetimi, SSL/HTTPS Konfigürasyonu
* **SEO:** Google Search Console, XML Sitemap, Meta & OG Tag Management
